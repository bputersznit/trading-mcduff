# DAILY TRADING CHECKLIST - NT8 SURVIVAL MODE

## PRE-MARKET (Before Trading Starts)

### Account Status Check
- [ ] **Current Cumulative P&L**: $______
  - If < -$400: Consider taking day off
  - If < -$450: STOP trading until reset
  - If = -$500: ACCOUNT LOCKED (NT8 breach)

- [ ] **Last 5 Days P&L**: $______
  - Add: Day -5, Day -4, Day -3, Day -2, Yesterday
  - If ≤ -$250: **DO NOT TRADE TODAY** (weekly limit)
  - If between -$200 to -$249: High alert, reduce size/frequency

### System Check
- [ ] NT8 connection active
- [ ] Order flow data feed working
- [ ] All three risk management layers enabled:
  - [ ] Rolling 4 tr/hr gate: ON
  - [ ] Weekly -$250 limit: CHECKED (see above)
  - [ ] Hard -$500 stop: ON
- [ ] Position size: **1 CONTRACT ONLY**

### Strategy Parameters Verified
- [ ] ABSORPTION: 6pt target, 3pt stop, 2min max
- [ ] ICEBERG: 5pt target, 2.5pt stop, 1.5min max
- [ ] BREAKOUT: 8pt target, 4pt stop, 1min max

## DURING TRADING

### Every Hour
- [ ] Check trades in last 60 minutes
  - If < 4 trades/hour: System should stop (verify gate working)
  - If > 15 trades/hour: Check for over-trading

### After Every Trade
- [ ] Log result: Win/Loss/Scratch
- [ ] Update session P&L
- [ ] Check if approaching limits:
  - Session P&L < -$100: Increase caution
  - Session P&L < -$150: Consider stopping

### Warning Signs - STOP IMMEDIATELY IF:
- [ ] 3 consecutive losses > $10 each
- [ ] Any single loss > $50 (something wrong)
- [ ] Win rate < 30% after 10+ trades
- [ ] Slippage consistently > $2/contract
- [ ] Emotional/revenge trading urges

## EMERGENCY PROCEDURES

### If Weekly Limit Hit (-$250 in 5 days)
1. **STOP trading immediately**
2. Do NOT trade for rest of week
3. Review what went wrong
4. Check if strategy still valid
5. Resume next Monday (fresh 5-day window)

### If Approaching Hard Limit (< -$400 cumulative)
1. **Take 3 days off**
2. Paper trade only
3. Review ALL trades for last month
4. Identify systematic issues
5. Consider reducing frequency or taking longer break

### If Hard Limit Breached (-$500)
1. **STOP all automated trading**
2. NT8 privileges lost
3. Full strategy review required
4. Manual trading only (if allowed)
5. Need to rebuild capital before re-automating
